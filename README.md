# DineBuddy 🍽️

DineBuddy is a restaurant table booking web application built using Django.

## Features

- User registration and login
- Restaurant browsing
- Table booking system
- Time slot management
- Owner dashboard
- Restaurant analytics
- Offers and coupons
- Menu management

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- Chart.js
